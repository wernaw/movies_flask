# Flask Movie Manager

A simple Flask web application to manage a list of movies. Users can add movies with title, year and actors,
view the list of movies and remove movies.

---

## Features

- View all movies
- Add new movies
- Remove existing movies
- Flash messages for feedback (success/error)

---

## Requirements

- Python 3.8+
- Flask

---

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/wernaw/movies_flask
cd <your-repo-directory>
```

2. **Install dependencies:**

```bash
pip install -r requirements.txt
```
## Usage

1. **Run the Flask app:**
```bash
python app.py
```

2. **Open your browser and go to:**

```bash
http://127.0.0.1:5000/
```

3. **Open your browser and go to:**
- Add movies via the add movie page.
- Remove movies using checkboxes on the home page.
- View all movies on the home page.
